AI Comment Classifier
=====================

This is a basic AI agent tool created as a practice project. It categorizes user comments into six categories: negative feedback, bug report feedback, feature request feedback, other feedback, general suggestion feedback, or positive feedback. This helps companies organize feedback so they can address bugs promptly, respond to suggestions, and note positive feedback appropriately.

How to Use
----------

1.  Submit a comment via the provided form or input method.
    
2.  The AI agent will process the comment and classify it into one of the six categories.
    
3.  Once classified, comments are routed to the appropriate section for follow-up: bugs are prioritized, suggestions are noted, and positive feedback is acknowledged without unnecessary urgency.
